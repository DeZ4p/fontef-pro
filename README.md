# FontEF Pro

Professional Font Manager Module for KernelSU with WebUI

## Features
- Universal font replacement for ALL Android devices
- Beautiful WebUI with 8 languages
- Auto dark/light theme
- meta-overlayfs support
- Works on Xiaomi, Samsung, OnePlus, Pixel, Huawei, TECNO, and more
- No need to reinstall module for updates

## Installation
1. Download latest ZIP from [Releases](../../releases)
2. Install via KernelSU
3. Reboot
4. Open KernelSU → Modules → FontEF Pro → WebUI

## Supported Devices
- Xiaomi / Poco / Redmi (MIUI, HyperOS)
- Samsung (One UI)
- OnePlus / Oppo / Realme (OxygenOS, ColorOS)
- Google Pixel (Pixel UI)
- Huawei / Honor (EMUI, HarmonyOS)
- TECNO / Infinix
- AOSP / Custom ROMs (LineageOS, crDroid, PixelOS)
