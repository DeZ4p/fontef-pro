## 📥 Installation

1. Download the latest ZIP from [Releases](../../releases)
2. Open KernelSU app
3. Install module (flash ZIP)
4. Reboot your device
5. Open KernelSU → Modules → FontEF Pro → WebUI
6. Upload a font file (.ttf or .otf)
7. Tap Apply Font
8. Reboot again
